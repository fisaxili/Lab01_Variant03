# Задачи

## Задача 1 (базовая)
Перевести температуру из градусов Цельсия в Фаренгейты. Формула: F = C × 1.8 + 32.


## Задача 2 (средняя)
Перевести температуру из Цельсия в Кельвины. Формула: K = C + 273.15.


## Задача 3 (расширенная)
Вычислить разницу между температурой в Фаренгейтах и Кельвинах для заданной температуры в Цельсиях. Определить, какая шкала даёт большее числовое значение.

Формулы:
- Перевод температуры из Цельсия в Фаренгейты: F = C × 1.8 + 32
- Перевод температруы из Цельсия в Кельвины: K = C + 273.15

## Системные требования

- Платформа: .NET 8.0 Runtime или выше
